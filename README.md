### Notes Web App

> Uses Tauri, Next JS, Surreal DB

> Built with Rust, TypeScript, Tailwind CSS

### Development
```bash
### clone the repo
git clone https://github.com/nodecodeapp/NodeCodeDesktop.git -b dev

### install dependencies
npm install

### run the app
npm run tauri dev
```

### Build
```bash
### build frontend
npm run build
### build the app
npm run tauri build
```
