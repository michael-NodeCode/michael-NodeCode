### NodeCode v1.3.0

> Uses Tauri, Next JS, Rusqlite, and Tailwind CSS

### Development
```bash
### clone the repo
git clone https://github.com/nodecodeapp/NodeCodeDesktop.git -b dev

### change directory
cd NodeCodeDesktop

### install dependencies
npm install

### run the app
npm run tauri dev
```

### Build
```bash
### build frontend
npm run build
### build the app
npm run tauri build
```
